pgsampler

statistics shipping background worker for Postgresql


Source Code     Documentation

Ship Metrics to Pgantenna

Pgsampler can be configured to send statistics to a pgantenna instance for for monitoring, alerting, and analysis.

Log Metrics to CSV

Pgsampler can be configured to log a history of performance statistics in flat files for later analysis.

MiniConf

Pgsampler is a background worker that starts and stops with your Postgresql cluster.

It requires minimal configuration to start collecting data.
